- **Gate Claude workout analysis on saved context** - FIT import no longer auto-triggers Claude classification, and `POST /api/training/workouts/{id}/analyze` now returns 409 `workout_context_required` until the user saves a `workout_context` row (surface, run type, HR source, feel notes, planned speed plan). Once saved, the prompt builder injects those fields and — for treadmill workouts — overrides device pace with the resolved planned speed (expanding `repeats` and `same_as_previous`). The nightly stride evaluation also folds feel notes plus a structured plan summary into the per-workout notes sent to Claude. (Hytte-6u5k)
