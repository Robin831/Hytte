category: Added
- **Family Chat schema and store** - Introduced the `family_chat_conversations`, `family_chat_members`, and `family_chat_messages` tables (created at startup) plus an internal store package with helpers for creating conversations, listing/reading conversations a user belongs to, appending messages, paginating message history, and marking conversations read. Conversation names, message bodies, and attachment paths are encrypted at rest; non-member access returns a forbidden error. The `family_chat` feature flag remains off by default — HTTP handlers and frontend arrive in follow-up beads. (Hytte-0w3d)
