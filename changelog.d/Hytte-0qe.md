- **Dashboard quick links/bookmarks widget** - Authenticated users can save, remove, and manage personal bookmark links stored as a user preference. Links persist via the `quick_links` preference key. Includes URL validation (http/https only) to prevent XSS, a 50-link limit, max length validation (200 chars for title, 2048 for URL), mobile-accessible remove buttons, and auth-gated rendering.
