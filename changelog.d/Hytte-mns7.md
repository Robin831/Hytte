category: Added
- **Forge Dashboard: full queue view with bead labeling** - Adds a Full Queue card to the Forge Dashboard showing all beads (ready, in-progress, unlabeled, needs-attention) grouped by anvil with collapsible sections. Each bead displays its ID, title, priority, and labels. Labels can be added or removed directly from the dashboard via IPC commands (`label-add`/`label-remove`), enabling forge queue management without SSH access. New backend endpoints: `GET /api/forge/queue/all`, `POST /api/forge/beads/{id}/labels`, `DELETE /api/forge/beads/{id}/labels/{label}`. (Hytte-mns7)
