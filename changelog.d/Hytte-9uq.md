category: Added
- **Infrastructure monitoring modules with SSRF protection** - Added three built-in infra modules: Service Health Checks (HTTP endpoint monitoring), SSL Certificate Expiry (TLS cert expiration tracking), and Uptime History (historical uptime statistics). Includes API endpoints for managing monitored targets and viewing check results. URLs and hostnames are validated at both input time and connection time using a safe dialer that blocks private/internal IPs and prevents DNS rebinding. HTTP redirects are not followed to prevent redirect-based SSRF. Port range is validated for both SSL cert monitoring and health check service URLs. (Hytte-9uq)
