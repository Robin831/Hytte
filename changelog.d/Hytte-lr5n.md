- **Homework send message with streaming** - New POST endpoint for sending messages in homework conversations. Parses multipart form data with text and optional image upload, detects subject from message content, builds age-calibrated system prompts, and streams Claude responses back via SSE with real-time text deltas. Uploaded images are sent to Claude for analysis and stored permanently for conversation history. Auto-detected subjects are encrypted at rest. Supports session resumption for multi-turn conversations. (Hytte-lr5n)
