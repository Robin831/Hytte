category: Changed
- **Analyze button also produces Insights** - Clicking Analyze on the workout detail page now generates the richer Insights output (TrendCard, EffortSummary, SuggestionsCard, RiskFlagsSection, ConfidenceIndicator, threshold context) in addition to the existing tag analysis. Insights generation is gated on `claude_ai`; the `ai_auto_analyze` preference continues to gate only the upload-time auto-trigger, since the manual click is explicit consent. Cached Insights are reused without a new Claude call, and a failed Insights run no longer fails the request — the tag analysis is still returned. The frontend refetches `/insights` after Analyze so the new cards render without a page reload. (Hytte-a8eb)
