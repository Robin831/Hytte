- **Grid rectangle detector + browser crop pipeline** - Added `detectGrid()` and `cropCellsToCanvases()` to `web/src/components/pokemon/rectangleDetector.ts`. The detector reuses the existing Sobel pass over a downsampled frame, falls back to a percentile-based page bounding box, buckets edges into a uniform rows×cols grid, and accepts cells whose per-bucket bbox lands inside the 5:7 card aspect window — tolerating empty pockets and up to ~5° of page skew. The crop helper rescales each accepted cell into a 1500×2100 canvas for ingestion. Internal foundation for the upcoming PageScanner; no UI yet. (Hytte-ku2w)
