- **Budget: CSV import for migration** - Added `POST /api/budget/import/csv` (preview) and `POST /api/budget/import/csv/commit` endpoints for importing transactions from CSV files. Handles Norwegian number formats (comma decimal, space thousands), multiple date formats (ISO, DD.MM.YYYY, etc.), and column mapping. The import flow has a two-step review-then-commit design so users can inspect parsed rows before saving. Includes a `/budget/import` frontend page with file upload, column mapping UI, and a filterable preview table. (Hytte-8yf2)
