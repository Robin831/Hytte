category: Added
- **Link recurring transactions to variable bills** - Recurring rules can now be linked to a variable bill via a new `variable_id` field. When linked, the recurring amount is driven by that month's variable bill entries instead of the fixed amount — the regning (bill split) calculator uses the variable total, and generated transactions use the variable amount. The regning page shows the variable bill name with a link to the variables page for editing, and displays a warning when no entries exist for the current month. The recurring transaction form includes a dropdown to select a variable bill. (Hytte-2p2f)
