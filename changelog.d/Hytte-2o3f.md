- **Scroll-to-adjust on time input** - Mouse wheel now adjusts the time picker value when the input is focused and the dropdown is closed: scroll up = +1 min, scroll down = −1 min, Shift+scroll = ±15 min. Value is clamped to 00:00–23:59 and page scroll is suppressed while adjusting. (Hytte-2o3f)
