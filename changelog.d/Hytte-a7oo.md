category: Added
- **Work Hours: data model and calculation engine** - Adds the `work_hours` feature flag and `internal/workhours` package with SQLite schema for work days, sessions, deductions, and deduction presets. Includes a calculator that computes gross minutes, net minutes (after lunch and custom deductions), reported hours rounded down to the nearest configurable interval (default 30 min), remainder, and flex pool accumulation. REST API endpoints cover daily CRUD, session and deduction management, preset management, week/month summaries, and flex pool with manual reset. User settings for standard day, rounding increment, and lunch duration are stored in `user_preferences`. Sensitive fields (notes and deduction names) are encrypted at rest. (Hytte-a7oo)
