category: Fixed
- **Cards with no Cardmarket price now show "—" instead of "kr 0"** - pokemontcg.io returns an empty `cardmarket` block for cards in the new Mega Evolution series (me1, me2, me2pt5, me3), so our sync stored `price_eur=0`. The UI rendered that as "kr 0,00", making cards look free. The frontend price formatters in `CardLightbox`, `AddCardPanel`, `ScanDetailModal`, `PokemonTop`, and `PokemonSet` now treat amount=0 the same as missing (Cardmarket never quotes at exactly €0,00, so 0 always means upstream is missing data). The daily `RefreshPrices` job will fill these in automatically once pokemontcg.io's scraper catches up. (Hytte-bivr)
