- **Pokémon quick-add card flow** - A floating "+" button on the Pokémon section opens a search dialog (modal on desktop, bottom sheet on mobile) where you can type a collector number ("025/195") or name fragment ("pika") and pick a card from the autocomplete results. Each result shows a thumbnail, card name, collector number, set name, and NOK price. Cards with a single variant are added to your collection immediately; cards with multiple variants prompt a small variant picker (Holo / Reverse Holo / Normal) before adding. Escape and outside-click close the dialog, and a toast confirms each successful add. (Hytte-tceq)
