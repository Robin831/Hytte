category: Changed
- **Locale-aware date, time, and number formatting** - All date, time, and number displays across the app now respect the selected language via `i18n.language` using Intl APIs. Replaced all `toLocaleDateString(undefined)`, `toLocaleTimeString(undefined)`, `toLocaleString(undefined)`, and `toFixed()` display calls with the shared `formatDate`, `formatTime`, `formatDateTime`, and `formatNumber` utilities. Added `formatDateTime` utility for combined date+time formatting. Norwegian numbers now use space thousands separator and comma decimal; Thai dates show Gregorian calendar year. (Hytte-fs4o)
