category: Added
- **Pokémon sets browser** - New `/pokemon` page lists every set grouped by series. Scarlet & Violet, Sword & Shield, and Sun & Moon sections are expanded by default; older eras are tucked behind a "Show older sets" toggle. Each tile shows the set logo, name, release date, total cards, and the user's ownership progress (e.g. "12 / 195 — 6%"). The `/api/pokemon/sets` endpoint now returns an `owned_count` field so the page renders ownership without a follow-up request per set. A placeholder route at `/pokemon/sets/:id` is wired up until the set-detail page lands. (Hytte-jw6y)
