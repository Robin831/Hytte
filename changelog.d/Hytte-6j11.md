category: Added
- **Skeleton component** - Added `Skeleton` UI component (`web/src/components/ui/skeleton.tsx`) following the shadcn/ui pattern, providing animated placeholder loading states for content. Defaults to `aria-hidden` when no label or children are provided; auto-exposes when `aria-label`, `aria-labelledby`, or children are present. All Skeleton loading states across Admin, AllowancePage, Family, FamilyChildDetail, LactateInsights, LactateNewTest, LactateTestDetail, LactateTests, Links, MyChoresPage, Notes, Settings, Stars, Transit, Weather, Webhooks, WorkHoursPage, NetatmoWidget, and WeatherWidget now include `role="status"`, `aria-live="polite"`, and `sr-only` loading text using the `common:skeleton.loading` i18n key or existing namespace-specific loading keys.
