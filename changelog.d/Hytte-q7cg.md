category: Changed
- **Pokémon: hybrid sync overlays TCGdex prices for Mega Evolution sets** - pokemontcg.io v2's Cardmarket bridge doesn't carry the new Mega Evolution series (me1, me2, me2pt5, me3, me4 — ~737 cards combined), so those cards rendered as "—" since Hytte-bivr. SyncCards and RefreshPrices now dispatch a TCGdex price overlay for those specific sets — card metadata (id, name, rarity, images) continues to come from pokemontcg.io, but Cardmarket prices come from TCGdex (`api.tcgdex.net`, free, no auth). Per-card 404 errors (common for sets TCGdex hasn't ingested yet, like me4 on release day) are logged at the file level and skipped — affected cards stay at "—" until the next refresh picks them up. The fallback set list is a small hardcoded allowlist; expansion is one line each time a new pokemontcg.io gap surfaces. Follow-up Hytte-tuax tracks the eventual full migration off pokemontcg.io. (Hytte-q7cg)
