category: Added
- **Achievement unlock overlay with fanfare** - Regnemester now shows a full-screen celebration overlay when a Marathon or Blitz run unlocks one or more achievements. Each unlock plays a fanfare cue, fires a haptic pulse, and queues follow-up unlocks so multi-unlock runs animate sequentially. Dismissible by tap, Enter, Space, or Escape, and respects prefers-reduced-motion. Focus moves into the dialog on open, is trapped inside while open, and returns to the triggering element on close.
