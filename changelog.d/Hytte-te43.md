category: Added
- **Pokémon set detail page** - New `/pokemon/sets/:id` page renders every card in a set as a grid with owned/missing indicators, the current ownership count, and the total NOK value of owned variants. Each tile shows the card image, name, collector number, and the default variant's NOK price. Click a tile to open an inline detail panel where you can pick a variant (normal or reverse holo), set quantity and condition, add notes, and mark the card as owned. Filter the grid to "All", "Owned only", or "Missing only". Marking and unmarking are optimistic with toast feedback and revert on API errors. (Hytte-te43)
